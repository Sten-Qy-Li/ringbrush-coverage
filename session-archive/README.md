# Session Archive

This directory stores human-readable session-context notes for significant work
sessions on `ringbrush-coverage`. Each file summarises one session: its goal,
the key decisions made, the commits involved, and any context that future
sessions (human or AI) should know before continuing related work.

## File naming convention

```
YYYY-MM-DD_short-topic-slug.md
```

Example: `2026-04-20_dead-reckoning-recalibration.md`

## What belongs here

- A brief narrative of what was attempted and why.
- Before/after states of algorithmic parameters or other non-obvious tuning.
- The git SHAs that bracket the change (so old artefacts remain traceable).
- Data sources used (paths, file names) that are external to the repository.
- Open questions or follow-ups that a future session should be aware of.

## What does NOT belong here

- Full conversation transcripts.
- Code snippets that duplicate the repository itself (link to files instead).
- Personal or credential information.
